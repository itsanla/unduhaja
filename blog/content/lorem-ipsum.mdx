## Apa Itu Lorem Ipsum?

**Lorem Ipsum** adalah teks placeholder yang telah digunakan dalam industri percetakan dan desain grafis sejak abad ke-16. Teks ini berasal dari karya *"de Finibus Bonorum et Malorum"* (Tentang Batas Kebaikan dan Kejahatan) yang ditulis oleh **Cicero** pada tahun 45 SM.

Teks Lorem Ipsum yang kita kenal saat ini sebenarnya adalah versi yang telah dimodifikasi dari teks asli Latin. Kata-kata tertentu telah diubah atau ditambahkan sehingga tidak lagi bermakna secara gramatikal.

## Mengapa Lorem Ipsum Digunakan?

Ada beberapa alasan utama mengapa Lorem Ipsum menjadi standar dalam industri desain:

1. **Distribusi huruf yang natural** — Lorem Ipsum memiliki distribusi huruf yang mirip dengan bahasa Inggris, sehingga layout terlihat realistis.
2. **Tidak mengganggu fokus** — Tidak seperti teks bermakna, Lorem Ipsum memungkinkan reviewer fokus pada elemen visual desain.
3. **Standar industri** — Penggunaannya yang sudah berabad-abad membuatnya mudah dikenali sebagai placeholder.

### Kode Contoh Generator Sederhana

```javascript
function generateLorem(paragraphs = 1) {
  const words = ['lorem', 'ipsum', 'dolor', 'sit', 'amet'];
  let result = '';
  for (let i = 0; i < paragraphs; i++) {
    const length = Math.floor(Math.random() * 20) + 10;
    const paragraph = Array.from({ length }, () => 
      words[Math.floor(Math.random() * words.length)]
    ).join(' ');
    result += paragraph + '.\n\n';
  }
  return result;
}
```
