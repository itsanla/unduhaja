## 2. Automasi Tugas Berulang

> "Jika Anda melakukan sesuatu dua kali, automasi pada yang ketiga." â€” Prinsip developer produktif

### Script Automation

```bash

# Script untuk setup project baru
create_project() {
  mkdir -p "$1"/{src,tests,docs}
  cd "$1"
  git init
  npm init -y
  echo "node_modules/" > .gitignore
  echo "Project $1 siap!"
}

create_project my-awesome-project
```
